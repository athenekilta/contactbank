# Contact bank

## Dev dependencies

Must have Gulp installed.

``` bash
npm install -g gulp
```

## Usage
``` bash
git clone https://github.com/coumbole/contactbank.git
cd contactbank/gulp
npm install
gulp
```

A new browser window should open at localhost:3001.
